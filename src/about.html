Om sidan

Publicering

Webbplatsen är publicerad via publicerings-plattformen Azure för studenter och det är första gången jag testar att den här plattformen. Azure fungerar för flera olika programmeringsspråk som till exempel JavaScript, Node.js, Python, C#, PHP och .NET.
För databaser finns det stöd för både relationsdatabaser som MariaDB, MySQL, PostgreSQL och för NoSQL som Cosmos DB. Azure Student är gratis till en viss gräns och man får ett visst "belopp" att använda.
För att publicera till Azure använder jag GitHub tillsammans med Azure App Services för att deploya koden. Varje gång jag gör en build och push till Github-repot uppdateras webbplatsen, och publiceras, via CI/CD-pipelines.

Fördelar

En fördel med Azure är att det finns väldigt många möjligheter, val och funktionerna som gör det möjligt att skapa en bredd av applikationer/program. Azure stödjer även automatisering och CI/CD vilket gör det väldigt smidigt att använda med GitHub-repos och uppdateringar av koden på ett enkelt sätt.

Nackdelar

Nackdelen med Azure, enligt mig som nybörjare, är att det är så mycket val och funktioner så det blir lite överväldigande och svårt att veta/navigera hur man ska gå till väga.
